# BuscoTattoo

Plataforma web y móvil para encontrar tatuadores por categoría de estilo y ubicación geográfica, vinculando sus perfiles de Instagram.

## Stack
- Web: React + Vite + TypeScript
- Mobile: React Native + Expo (fase 2)
- Backend: Node.js + Express + TypeScript
- Base de datos: PostgreSQL
- Monorepo: Turborepo
